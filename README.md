# FinFlow — Finance Dashboard

Premium personal finance dashboard built with **Next.js 14 (App Router)** + **JavaScript**.

## Quick Start

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Deploy to Vercel

```bash
npx vercel
```

Or push to GitHub → import at vercel.com/new (auto-detects Next.js).

## Stack

- **Next.js 14** — App Router
- **Tailwind CSS** — Styling + dark mode
- **Recharts** — Area chart + Donut chart
- **Lucide React** — Icons
- **next-themes** — Dark / Light mode
- **Context API** — State management
- **localStorage** — Persistence

## Features

- [x] Dashboard with animated metric cards (count-up animation)
- [x] Balance Trend Area Chart with 7D / 14D / 30D toggle
- [x] Spending Breakdown Donut Chart (hover interactions)
- [x] Transactions table — search, filter, sort (3 columns)
- [x] Add / Edit / Delete transactions (Admin only)
- [x] Role switcher: Admin ↔ Viewer (Navbar dropdown)
- [x] Smart Insights with observations
- [x] Dark Mode + Light Mode toggle
- [x] localStorage persistence
- [x] Responsive — collapsible sidebar on mobile
- [x] Staggered entrance animations on all pages
- [x] Empty state UI for no results
